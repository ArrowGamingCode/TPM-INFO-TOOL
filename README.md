# TPM-INFO-TOOL

An experimental tool that displays technical information to help troubleshoot TPM-related settings for gaming.

## 📦 How to Use

1) Download and run 'TPM INFO TOOL.cmd'. You will see a UAC prompt.<br />
2) The program will display a list of TPM related info and automatically copy it to your clipboard.
3) Look for the overall TPM Attestation PASS/FAIL at the bottom!
